# Pokedex

## Description

Pokedex - parte del curso de Nest

## Ejecutar en desarrollo

1. clonar el repositorio
2. Ejecutar

```bash
    $ yarn install
```

3. Tener Nest CLI instalado

```bash
    $ npm i -g @nestjs/cli
```

4. Levantar la base de datos

```bash
    $ docker-compose up -d
```

5. Reconstruir la base de datos semilla

```
http://localhost:3000/api/v2/seed
```

## Running the app

```bash
## development
$ yarn run start

## watch mode
$ yarn run start:dev
```

## Stack

- Mongo
- Nest
